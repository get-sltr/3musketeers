### Required Steps:
1. ✅ Run `npm run backup:full`
2. ✅ Run `npm run validate:pre`
3. ✅ Document what you're changing
4. ✅ Get approval (if changing core files)
5. ✅ Make changes
6. ✅ Run `npm run validate:post`
7. ✅ Test manually
8. ✅ Run `npm run build` to verify
